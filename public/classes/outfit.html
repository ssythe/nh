Class Outfit

This is used for setting player & bot body colors + assets.

Using the methods of this class will NOT replicate the player changes to connected clients. But will change their values on the server.

To replicate the changes use the Outfit.set method, or alternatively use:
the Player.setOutfit and Bot.setOutfit method.

example
Game.on("playerJoin", (p) => {
     p.on("avatarLoaded", () => {
         let outfit = new Outfit(p)
              // Sets all player colors to "#ffffff"
             .body("#ffffff")
             // Sets the head color (body colors are still changed!)
             .head("#000000")
             // Replicate the changes to the players.
             .set()

         // Alternatively, you can use: p.setOutfit(outfit)
     })
})
